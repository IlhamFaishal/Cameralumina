## âœ¨ Features

- **Cinematic Scroll Sequence**: Frame-by-frame HTML Canvas scroll animation (Apple-style) for the hero section.
- **Immersive Storytelling**: Text reveals and parallax effects powered by **GSAP ScrollTrigger**.
- **Interactive 3D Cards**: Glassmorphism feature cards with tilt effects and premium hover states.
- **Horizontal Gallery**: Smooth horizontal scroll gallery pinned to the viewport.
- **Premium Aesthetics**: Dark luxury theme, noise textures, custom typography (Inter + Playfair Display), and refined spacing.
- **Performance Optimized**: Built on Next.js App Router with TurboPack.
